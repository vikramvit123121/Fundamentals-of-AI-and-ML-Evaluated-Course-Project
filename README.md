This project is a machine learning application designed to predict a student's academic performance (Final Marks) based on three key behavioral features: Study Hours, Sleep Hours, and Attendance Percentage.
# Student Marks Predictor

## Overview

This is a simple Python project that predicts student marks based on:

* Study hours
* Sleep hours
* Attendance percentage

It uses a Linear Regression model.

---

## Requirements

Make sure you have Python installed, then install these libraries:

pip install numpy pandas matplotlib scikit-learn

---

## How to Run

1. Download or copy the Python file
2. Open terminal or command prompt
3. Run the script:

python filename.py

4. Enter the inputs when prompted:

* Study hours
* Sleep hours
* Attendance

5. The program will display predicted marks and the plot using linear regression

---

## Example

Enter study hours: 6
Enter sleep hours: 7
Enter attendance (%): 80

Output:
Predicted Marks: 72.50

* Simple and easy to understand
* Uses multiple input factors
